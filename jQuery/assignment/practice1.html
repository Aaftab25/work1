What is jQuery?

jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript on your website. jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code.It is also very useful to simplify a lot of the complicated things from JavaScript, like AJAX calls and DOM manipulation.It is very fast and extensible.It facilitates the users to write UI related function codes in minimum possible lines.It improves the performance of an application.Browser's compatible web applications can be developed.It uses mostly new features of new browsers.
